Quick Reference

Slots (d1-d9):
d1(s("bd sd")) - slot 1
d2(note("c3")) - slot 2
d1(silence()) - mute slot

Patterns:
s("bd sd hh") - samples
note("c3 e3") - notes
.fast(2) - speed x2
bd*4 - repeat 4x
~ - rest
hush() - stop all

Effects:
.gain(0.7) - volume
.room(0.5) - reverb
.delay(0.3) - delay
.lpf(800) - filter
.pan(0.5) - pan

ALGORAVE - Help & Examples

🎵 Samples Disponibles (Drums)

bd - Bass Drum / Kick
sd - Snare Drum
hh - Hi-Hat
cp - Clap
oh - Open Hi-Hat
kick - Bass Drum
snare - Snare Drum
hihat - Hi-Hat
clap - Clap
openhh - Open Hi-Hat

🎹 Synthétiseurs Disponibles

sine - Onde sinusoïdale douce
square - Onde carrée (8-bit)
sawtooth - Onde dent de scie
triangle - Onde triangle
fm - Synthèse FM (modulaire)
am - Synthèse AM

📝 Syntaxe Mini-Notation

bd sd hh sd 4 sons espacés également
bd*4 Répéter 4 fois
bd ~ sd ~ ~ = silence/rest
<bd sd hh> Alternance chaque cycle

🎛️ Transformations de Patterns

.fast(2) Doubler la vitesse
.slow(2) Ralentir de moitié
.rev() Inverser le pattern
.every(4, fast(2)) Tous les 4 cycles

🎚️ Effets Audio

.gain(0.5) Volume (0-2, défaut 1.0)
.room(0.5) Reverb (0-1)
.delay(0.3) Delay/Echo (0-1)
.lpf(1000) Filtre passe-bas (Hz)
.pan(0.7) Panoramique (-1 à 1)

💡 Exemples Complets

Techno Basique 135 BPM

d1(s("bd*4"))
d2(s("hh*8"))
d3(s("~ sd ~ sd"))
d4(note("c2 ~ eb2 ~").s("fm"))

Psytrance Psychédélique

d1(s("bd*4"))
d2(s("hh*16").fast(2))
d3(note("c3 d3 eb3 g3").s("fm").room(0.5))
d4(s("oh ~ oh ~").delay(0.3))

Drum & Bass 174 BPM

d1(s("bd ~ sd bd ~ sd*2 bd ~"))
d2(s("hh*8"))
d3(note("c2 ~ c2 eb2").s("sawtooth").lpf(800))

Patterns Avancés

// Stack multiple patterns
d1(stack(
  s("bd*4"),
  s("hh*8"),
  note("c2").s("fm")
))

// Transformations conditionnelles
d2(s("bd sd hh cp").every(4, rev()))

// Effets combinés
d3(note("c3 eb3 g3").s("triangle")
   .room(0.4).delay(0.2).pan(0.7))

Contrôle des Slots

// Arrêter un slot spécifique
d1(silence())

// Arrêter tous les patterns
hush()

// Chaque slot est indépendant
d1(s("bd*4"))  // Joue dans slot 1
d2(s("hh*8"))  // Joue dans slot 2
d1(s("bd sd")) // Remplace le pattern du slot 1
